# 線上聖經教材導引

來源：FHL 多媒體聖經學苑手機入口 `https://fungclass.fhl.net/smartphone.html`
整理日期：2026-07-05

## 目的

FHL 的手機入口以圖片與舊式頁面為主，對手機使用者與產品整合不夠直覺。本文件把現有課程整理成「線上聖經教材導引」，讓使用者可以依照讀經階段快速找到合適教材，也方便未來在 App 內新增「教材」頁。

設計原則：教材是輔助讀經，不應干擾經文閱讀。入口應集中、分層、可掃描，並以外部連結開啟。

## 使用路徑

| 使用者狀態 | 建議入口 | 目的 |
| --- | --- | --- |
| 剛開始讀聖經 | 新舊約導讀 + 讀經方法 | 先建立聖經大架構與讀經方法 |
| 正在讀某一卷書 | 舊約單卷 / 新約單卷 | 補足書卷背景、結構與重點 |
| 想建立信仰框架 | 神學類 | 系統整理信仰主題 |
| 想把信仰落地 | 實踐類 | 對接婚姻、教育、宣教、民間信仰等生活議題 |

## App 呈現建議

建議未來新增一個獨立的「教材」主頁，不放在側邊書卷目錄，也不塞進閱讀 toolbar。

資訊架構：

1. 教材首頁只顯示六個分類：開始讀經、讀經方法、舊約單卷、新約單卷、神學主題、信仰生活。
2. 每個分類使用可掃描的卡片列表：標題、短說明、外部連結。
3. 單卷教材可在書卷簡介頁底部顯示「延伸教材」一列，但預設不展開。
4. 所有教材連結用新分頁開啟，避免使用者離開閱讀器後迷路。
5. 不在經文正文中插入教材提示，避免破壞專注閱讀與默想節奏。

## 建議資料模型

```ts
type CurriculumItem = {
  category: 'overview' | 'method' | 'old-testament' | 'new-testament' | 'theology' | 'practice';
  title: string;
  subtitle?: string;
  url: string;
  testament?: 'old' | 'new';
  bookName?: string;
  level: 'intro' | 'book' | 'topic';
};
```

若要正式做成 App 功能，下一步建議新增 `public/curriculum-guide.json`，再建立 `MainCurriculum.tsx` 作為獨立主畫面。
